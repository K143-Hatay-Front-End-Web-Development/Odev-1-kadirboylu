### Palindromic Number

A palindromic number is a number that remains the same when its digits are reversed.

The largest palindrome made from the product of two n digit numbers:

- n = 1 ➡️ 9
- n = 2 ➡️ 9009
- n = 3 ➡️ 90909
- n = 4 ➡️ 99000099
- n = 5 ➡️ 990090099
